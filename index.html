ContACT.jl

Applied category theory for contact matrices in infectious disease modelling.

Overview

ContACT.jl provides a category-theoretic framework for constructing, manipulating, and composing age-structured contact matrices from social mixing survey data. Built on Catlab.jl, it provides formal guarantees (functoriality, idempotence, associativity) verified both in tests and companion Lean 4 proofs.

Key Features

OperationOperatorCategory-theoretic interpretation
Composition⊕Commutative monoid on Hom(G,G)
Stratification⊗Kronecker product with coupling
Coarsening↓Left Kan extension along surjection
SymmetrisationsymmetriseIdempotent endomorphism

Quick Example

using ContACT
using LinearAlgebra

# Define age groups and a contact matrix
partition = AgePartition([0, 18, 65])
pop = [1000.0, 3000.0, 500.0]
M = [2.0 1.0 0.5;
     1.0 3.0 1.0;
     0.5 1.0 1.5]
cm = ContactMatrix(M, partition, pop)
ContactMatrix{Float64, MeanContacts} (3×3 age groups)
# Symmetrise (idempotent endomorphism)
cm_sym = symmetrise(cm)
matrix(cm_sym)
3×3 Matrix{Float64}:
 2.0    0.666667  0.75
 2.0    3.0       3.5
 0.375  0.583333  1.5
# Coarsen to 2 groups (left Kan extension)
coarse = AgePartition([0, 18])
cm_coarse = cm ↓ coarse
matrix(cm_coarse)
2×2 Matrix{Float64}:
 2.0  0.928571
 1.5  3.78571
